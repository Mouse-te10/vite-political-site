# Функционал
  *Слайдер
  *Форма отправки
  *Адаптивность
  *Добавлена карта

#Технологии
  **React** - JS библиотека
  **HTML & CSS** - языки разметки и стилей
  **Vite** - сборщик проектов
  **Yandex maps** - добавление карты на сайт

#Инструкция по запуску проекта
  1. git clone https://github.com/Mouse-te10/vite-political-site.git
  2. npm i
  3. npm run dev
